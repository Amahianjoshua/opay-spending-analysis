# 💳 OPay Spending Account Analysis (Power BI Dashboard)

## 📌 Project Overview

This project explores my personal **OPay spending account transactions** to understand spending patterns, transaction channels, and financial behavior over time.

Using **Power BI**, I transformed raw bank statement data into an interactive dashboard that reveals key insights about how money flows in and out of the account.

> ⚠️ Note: This OPay account functions primarily as my **spending account**, not my main account.

---

## 🎯 Project Objective

The goal of this project was to answer a simple question:

**Where does my money actually go throughout the year?**

By analyzing transaction data and visualizing it in Power BI, I aimed to uncover patterns in:

- Spending categories
- Transaction channels
- Monthly financial activity
- Account balance behavior

---

## 📊 Key Insights

- 📱 **Mobile banking dominates transactions**  
  94% of all transactions were performed through mobile banking.

- 💸 **Transfers are the most frequent expense category**  
  A significant portion of spending occurs through transfers.

- 📅 **Seasonal transaction spikes**  
  Transaction activity increases notably in **April and December**.

- ⚡ **Fast cash flow behavior**  
  The account balance dropped to **₦0 at some points**, indicating that funds move in and out quickly.

## 💡 Recommendations

Based on the analysis:

- Track transfers more carefully since they represent a major portion of spending.
- Maintain a small buffer balance to avoid fully depleting the account.
- Monitor high-activity months such as **April and December** to prevent overspending.
- Continue using **data visualization tools like Power BI** to track financial behavior over time.
